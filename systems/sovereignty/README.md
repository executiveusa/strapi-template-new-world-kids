# Sovereignty System — ICM Stage Contracts

Impact/grant management workflow for New World Kids.
Owned by the **Sovereignty Navigator** agent role.

## Stages

| Stage | Directory | Purpose |
|-------|-----------|---------|
| 00 | 00_truth | Mission facts, evidence, Core Four records |
| 01 | 01_needs | Documented community needs |
| 02 | 02_opportunity_discovery | Funder research and opportunity mapping |
| 03 | 03_qualification | Go/no-go qualification per funder criteria |
| 04 | 04_relationship_mapping | Contact mapping and relationship status |
| 05 | 05_outreach | Draft outreach — requires Bambu approval before send |
| 06 | 06_application | Application drafts and submissions |
| 07 | 07_followup | Post-submission follow-up tracking |
| 08 | 08_awards_and_commitments | Awards, commitments, compliance schedule |
| 09 | 09_learning | Post-cycle retrospectives and lessons |

## Rules

- No agent may self-attest qualification for a grant
- No agent may send outreach without explicit Bambu approval
- All public claims must map to an approved story record or evidence document
- Grant Scout and Relationship Steward roles are read-only until Bambu approves an action
