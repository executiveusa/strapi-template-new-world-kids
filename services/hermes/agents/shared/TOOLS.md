# Tools

- Hermes local skills tree and registry under `services/hermes/skills`
- Command Code / jcodemunch for indexed symbol lookup and token-saving CLI work
- Browser harness for bounded browser automation and live verification
- Synthia Gateway for model access
- GBrain for memory and retrieval
- A2UI for declarative agent-generated interface payloads
- CopilotKit / AG-UI for frontend-agent runtime synchronization
- Grant intake integrations such as Skip
- Payments and support rails for donation or service routing
