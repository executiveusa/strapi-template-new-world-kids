# Playbook

1. Start from public proof before writing persuasion.
2. Keep food, water, energy, and shelter as the core taxonomy.
3. Route tax-deductible giving to the nonprofit rail.
4. Keep services and charitable support clearly separated.
5. Require approval before moving money, filing grants, or communicating as a human.
6. Use A2UI for declarative agent UI and CopilotKit/AG-UI when live frontend-agent coordination is needed.
