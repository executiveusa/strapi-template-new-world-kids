# Cinematic Site Components

Source repo: https://github.com/robonuggets/cinematic-site-components.git

Use this repo as a motion-pattern reference when building the POST-MAXX homepage and cinematic story sections.

## Useful patterns

- Text mask reveal
- Sticky stack narrative
- Layered zoom parallax
- Horizontal scroll hijack
- Curtain reveal
- Scroll SVG draw
- Kinetic marquee
- View-transition morphing
- Gradient shift and ambient motion

## Practical rule

- Use these patterns to make the page feel like a continuous film, not a pile of separate sections.
