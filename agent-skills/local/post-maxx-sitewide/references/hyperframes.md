# Hyperframes Helper

Source repo: https://github.com/robonuggets/hyperframes-helper.git

Use Hyperframes when the output should be a deterministic HTML-to-MP4 motion piece.

## Best fit

- Storyboards before a final composition
- Website-to-video capture
- Guided videos with a real recording plus motion graphics

## Simple rule

- Use HTML, CSS, and GSAP for the motion source.
- Keep clip timing deterministic.
- Render to MP4 when the motion needs a finished video asset.

## Practical rule

- Use this when the user wants a polished cinematic clip, not an interactive page.
