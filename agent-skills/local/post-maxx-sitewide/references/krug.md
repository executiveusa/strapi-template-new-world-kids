# Steve Krug rules for owner-facing pages

Use these rules on any dashboard or management page.

## Keep it simple

- One page, one main job.
- Make the primary action obvious.
- Use plain language.
- Show status at a glance.
- Group related actions together.
- Remove unnecessary choices.

## What to avoid

- Jargon
- Clever labels
- Hidden actions
- Busy dashboards with no clear next step

## Practical rule

- If a business owner would have to think hard to know what to do next, the page is not done.
