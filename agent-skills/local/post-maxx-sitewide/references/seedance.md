# Seedance

Source skill: https://github.com/robonuggets/seedance-skill.git

Use Seedance when the work needs motion graphics or a cinematic short.

## Best fit

- Image-to-video for a single hero frame
- Reference-to-video for multiple screenshots or visual states
- Text-to-video for concept shots when no imagery exists yet

## Prompt rules

- Keep the prompt focused and short.
- Describe each reference image by number and role.
- Avoid text/logos in the generated clip prompt.
- Use explicit motion language, not vague style language.

## Practical rule

- Use Seedance for story beats, intro frames, and motion studies.
- Use the result as a reference or a clip, then polish in the app.
